TI1805-Project

###Werkwijze

Neem issue aan en maak een pull request als de implementatie van
jouw functionaliteit compleet is.

Hoe te compilen?!
=================

Unix-like (OSX / Linux): 

````
$ mkdir bin
$ cd bin
$ cmake ..
$ make
````

En dan uitvoeren met:
````
$ ../src/project
````
